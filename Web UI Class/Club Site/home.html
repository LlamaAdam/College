Hello and welcome to the website for the Kansas City Soccer Club. The club has various groups for all ages that you can join if you are interested. We do have some products if you are interested in supporting the club.
The club was created back in 1980 and started as a group of guys playing. It is now a large club that offers 6 different age groups for you to play in. We also have some training if you wish to learn more about the sport. Thank you for checking out our website.
