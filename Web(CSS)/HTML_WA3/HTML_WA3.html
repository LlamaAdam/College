The IEBGENER utility is one of IBM's first data set utilities. This utility will allow you to do the following things

  • copy data sets from any input device and put it on any other input device
  • allows input records to be edited before creating a new output
  • allows single sequential files to broken up into multiple PDS members
  • allows two records from a sequential data set to be combined into one record

    • There are two different types of IEBGENER's you can use one is the basic IEBGENER and the other is an advanced IEBGENER. The basic IEBGENER will allow you to copy any file from one input source to an output source of your choice. This is the template for a basic IEBGENER

    Templete example

    //STEP1 EXEC PGM=IEBGENER
    //SYSPRINT DD SYSOUT=*
    //SYSUT1 DD input data information
    //SYSUT2 DD output data information
    //SYSIN DD DUMMY
    //

    EXAMPLE #1

    basic IEBGENER this example prints out the numbers 1-10

    //STEP1 EXEC PGM=IEBGENER
    //SYSPRINT DD SYSOUT=*
    //SYSUT1 DD DATA
    0001
    0002
    0003
    0004
    0005
    0006
    0007
    0008
    0009
    0010
    //SYSUT2 DD SYSOUT=*
    //SYSIN DD DUMMY
    //

    The advanced IEBGENER will allow you to edit the input data set through a series of control cards. These cards are inserted after the //SYSIN DD card. Notice in this template that the //SYSIN DD DUMMY changes to a //SYSIN DD * this forces you to put in in-stream data this data will be the control cards you need to use.

    EXAMPLE #2

    advanced IEBGENER this example show how to add 5* to the end of each record

    //STEP1 EXEC PGM=IEBGENER
    //SYSPRINT DD SYSOUT=*
    //SYSUT1 DD DATA
    0001
    0002
    0003
    0004
    0005
    0006
    0007
    0008
    0009
    0010
    //SYSUT2 DD SYSOUT=*
    GENERATE MAXFIELDS=1,MAXLITS=5
    RECORD FIELD=(5,'*****',5)
    //
